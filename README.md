Smart house project for the _Embedded Software for the Internet of Things_ course @ University of Trento.
Project developed by Leonardo Tartini, Giovanni Todesco, Mikele Golemi, Francesco Zadra

*MAIN FEATURES*
- alarm system with motion sensors
- automated lighting system with motion sensors
- AI driven car plate recognizer for automatic gate 
- automated heating system
- infrared remote control on the whole house
- web application control on the whole house

*SENSORS & ACTUATORS*
- motion sensors
- temperature + humidity sensor
- passive buzzer
- LEDs
- IR receiver and remote control
- webcam
- servo motor
- potentiometer
